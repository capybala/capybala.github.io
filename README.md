capybala.github.io
==================

Website of capybala.com


開発に必要な最初の作業（一度だけ必要）
-------------------------------------

[Jekyll](http://jekyllrb.com/)のインストール

```
sudo gem install jekyll
```


開発用サーバーの起動
--------------------

```
cd ~/capybala.github.io
jekyll serve --watch
```

この状態で、ブラウザで http://localhost:4000/ にアクセスすると表示される。

サーバーの終了はCtrl+C


公開
----

GitHubにプッシュ（Sync）すると公開される。
